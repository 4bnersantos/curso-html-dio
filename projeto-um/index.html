O que são Tags

As tags HTML são instruções que informam ao navegador como estruturar e formatar o conteúdo de uma página da internet. Elas são os blocos de construção do HTML Hypertext Markup Language e estão presentes em todas as páginas da web.

O que são textos em HTML

No HTML, os textos são estruturados por meio de elementos e tags, que indicam ao navegador como apresentar o conteúdo:

Tags São marcadores que definem a estrutura semântica da página, ou seja, a organização das informações. As tags são identificadas por um sinal de menor que "<", seguido pelo nome da tag e um sinal de maior que "">".
Elementos São usados para delimitar ou agrupar diferentes partes do conteúdo. Por exemplo, o elemento "p" define um parágrafo, enquanto o elemento "title" define o título da página.
Atributos Definem como o conteúdo deve ser apresentado. Por exemplo, o atributo “href” determina o destino de um link.

O que são Listas Ordenadas e Não Ordenadas

A principal diferença entre listas ordenadas e não ordenadas em HTML é a forma como os itens são apresentados:

Listas ordenadas Os itens são enumerados, podendo ser por número, letra ou algarismos romanos. São usadas para indicar uma sequência ou numeração, como uma receita. A estrutura de uma lista ordenada é delimitada pelas etiquetas "OL" e "/OL".
Listas não ordenadas Os itens são apresentados com marcadores, como pequenos círculos pretos. São usadas quando a ordem dos itens não importa, como uma lista de compras. A estrutura de uma lista não ordenada é delimitada pelas etiquetas "UL" e "/UL".
O elemento HTML "li" é usado para representar um item que faz parte de uma lista. Ele deve estar contido em um elemento pai, que pode ser uma lista ordenada, uma lista desordenada ou um menu.

O que são Links

Links em HTML, também conhecidos como hiperlinks, são elementos que conectam um documento a outro na web ou a uma área específica de uma página

Exemplo: Se você clicar AQUI vai ser redirecionado para uma outra página.
